# Phase 8D - Home Jobs Result

Delivered:

- a dedicated active-generation rail on Home;
- progress, model/template and prompt context for running jobs;
- direct navigation from an active card to Result Viewer;
- recent results restricted to completed jobs with output assets;
- deterministic newest-first sorting and three-item limits.

The view is derived from the existing persisted generation history. Live push updates remain dependent on the backend notification phase.
