# AllAi Visual System V2

## Visual thesis

AllAi is a black editorial media gallery with Manrope typography, one neon action color and quiet utility surfaces.

## Content hierarchy

- Home: one dominant media hero, curated horizontal rails, recent work.
- Create: prompt first, controls second, model context last.
- Projects: compact search/filter header and dense media grid.
- Pricing: short image-led purchase flow with explicit package selection.

## Interaction thesis

- Media uses a short fade-in and restrained image overlay for legibility.
- Selection changes are immediate, high-contrast and do not move layout.
- Sheets and navigation use compact, predictable motion from platform widgets.

## Tokens

- Background: `#000000`.
- Primary surface: `#101012`.
- Secondary surface: `#18181B`.
- Divider: `#2A2A2E`.
- Primary text: `#F5F5F7`.
- Accent: `#E6FF00`.
- Typeface: Manrope only.
- Interactive/card radius: 8 px; command pills may remain fully rounded.

## Rules

- No gradients on routine panels.
- No nested cards.
- One accent per screen.
- Large display type is reserved for onboarding and the Home brand signal.
- Working-screen titles stay between 28 and 36 px.
- Images carry hierarchy; borders and shadows remain quiet.
